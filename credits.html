License

How to add a character - If you have added an Image of a reknowned personality then you have to provide a proof along with it that you have the legal right to use that particular image or you have to insert the image in such a way that it can't be traced back.
Music - If you are adding music to the background or some other music then you have to provide a proof that you have the legal right to use those tunes (Provide the link to the license on the site)

Note - The licences and the proofs have to be added to this page so that we can credit the sources for their help
